POSTAL 4: No Regerts

23 August, 2022

  • game
  • professional
  • unreal engine
  • C++

Overview

Postal 4: No Regerts is an open world multiplayer narrative FPS game set in the Postal universe. I worked as a Gameplay Programmer at Running With Scissors to bring Postal 4 from Early Access to a 1.0 Release.

Role

Between 2020 and 2022, I contributed to Postal 4’s development as a Gameplay Programmer, implementing features using C++ and Unreal Engine Blueprints. I worked closely with design, art, and audio teams to ensure features integrated smoothly into the final product. Through this process I implemented the following features:

  • Modular roller coasters
  • Vending machines
  • Slot machines
  • Audio effects engine
  • Footstep VFX
  • Fluid physics
  • Modular & extensible side quest framework
  • Modular & extensible rampage framework
  • Hint system
  • Designer tools
  • Various carnival mini games
  • Various mission elements
  • Boss related Puzzle Mechanics
  • Bug fixes

This role strengthened my ability to bridge creative and technical disciplines in a remote environment while deepening my understanding of Unreal Engine.
