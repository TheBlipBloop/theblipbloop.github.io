Overview

The Ravenfield Multiplayer Project (RFMP) is a community driven project to add a multiplayer (PvP) mode to the single player game Ravenfield. As the base game did not support mods at this point, RFMP exists as a multiplayer remake of the original game.

Between 2018 and up to its release in 2019, I led a small team of developers in creating RFMP. I kickstarted the project, handled all programming, and worked with other developers to integrate weapons, maps, and other features. I also managed a global network of (50+) volunteer testers to help track down bugs and ensure the project’s stability.

To support high player counts on low end networking hardware, this project features novel network optimizations based on player attention heuristics. Servers only send information to players that would be reasonably interested in them.
